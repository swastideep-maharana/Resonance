# Resonance - AI Voice Cloning & TTS Platform

Resonance is a premium, high-performance platform for AI-powered Text-to-Speech (TTS) and Voice Cloning. Built with the latest bleeding-edge technologies, it offers a sleek, professional experience for creating and managing lifelike AI voices.

## 🚀 Tech Stack

Resonance leverages a modern, type-safe stack designed for speed, scalability, and developer experience:

*   **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
*   **Frontend Logic:** [React 19](https://react.dev/)
*   **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/) (Modern engine with native cascade layer support)
*   **Authentication:** [Clerk](https://clerk.com/) (Multi-tenant & Organization support)
*   **API & State:** [tRPC v11](https://trpc.io/) + [TanStack Query v5](https://tanstack.com/query) (End-to-end type safety)
*   **Database:** [PostgreSQL](https://www.postgresql.org/) with [Prisma ORM](https://www.prisma.io/)
*   **Storage:** [AWS S3](https://aws.amazon.com/s3/) (Scalable audio asset management)
*   **Forms:** [TanStack Form](https://tanstack.com/form) (Headless, type-safe form logic)
*   **Monitoring:** [Sentry](https://sentry.io/) (Real-time error tracking)
*   **Billing/Monetization:** [Polar](https://polar.sh/) Integration
*   **Key Libraries:**
    *   `wavesurfer.js` - Dynamic audio visualization
    *   `nuqs` - Type-safe search parameter management
    *   `lucide-react` - Beautiful, consistent iconography
    *   `zod` - Schema-based validation
    *   `dicebear` - Dynamic avatar generation

## ✨ Key Features

*   **Professional Text-to-Speech:** High-quality TTS generation with full control over stability, clarity, and style.
*   **Instant Voice Cloning:** Clone any voice with just a few seconds of audio samples.
*   **Voice Explorer:** A comprehensive hub to discover and preview system and custom-cloned voices.
*   **Sleek Modern UI:** A premium "glassmorphism" aesthetic featuring smooth multi-layered shadows and subtle micro-interactions.
*   **Organization-First:** Built-in support for multiple teams and collaborative voice management.
*   **Real-time Previews:** Interactive waveform visualization for generated audio files.
